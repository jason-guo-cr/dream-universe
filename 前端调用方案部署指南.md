# 前端调用LLM方案部署指南

## 📌 方案说明

由于云开发个人版**所有云函数操作都限制3秒超时**，无法完成LLM交互（通常需要20-30秒），因此采用**小程序前端直接调用LLM API**的方案。

### ✅ 优势
- 小程序前端没有超时限制
- 可以等待LLM任意长时间响应
- 实现简单直接
- 不需要复杂的异步/定时器方案

### ⚠️ 注意事项
- API密钥会在前端代码中（但对个人项目完全可接受）
- 需要配置小程序request域名白名单

---

## 🚀 部署步骤

### 1. 配置API密钥

打开 `miniprogram/utils/ai-helper.js`，修改API密钥：

```javascript
const SILICONFLOW_API_KEY = "你的硅基流动API密钥";
```

**获取API密钥：**
1. 访问 [硅基流动控制台](https://cloud.siliconflow.cn/account/ak)
2. 注册/登录账号
3. 创建API密钥
4. 复制密钥并替换上面的值

---

### 2. 配置request域名白名单

#### 方式一：微信小程序后台配置（推荐）

1. 登录 [微信小程序后台](https://mp.weixin.qq.com/)
2. 进入 **开发** → **开发管理** → **开发设置** → **服务器域名**
3. 在 **request合法域名** 中添加：
   ```
   https://api.siliconflow.cn
   ```
4. 保存配置（注意：域名配置一个月只能修改5次）

#### 方式二：开发工具临时配置（仅用于开发测试）

1. 打开微信开发者工具
2. 点击右上角 **详情**
3. 找到 **本地设置**
4. 勾选 **不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**

**注意：** 方式二仅用于开发测试，正式发布必须使用方式一配置域名白名单。

---

### 3. 上传云函数

虽然LLM交互在前端，但仍需要云函数来保存数据到云数据库。

#### 使用微信开发者工具上传：

1. 打开微信开发者工具
2. 右键点击 `cloudfunctions/dreamFunctions` 文件夹
3. 选择 **上传并部署：云端安装依赖**
4. 等待上传完成

---

### 4. 初始化云数据库

1. 打开 [云开发控制台](https://console.cloud.tencent.com/tcb)
2. 选择你的环境
3. 进入 **数据库**
4. 创建集合 `dreams`
5. 设置权限：
   - **读权限**：仅创建者可读
   - **写权限**：仅创建者可写

或者使用小程序首次运行时自动创建（推荐）。

---

### 5. 配置云开发环境ID

打开 `miniprogram/app.js`，检查云开发环境ID配置：

```javascript
wx.cloud.init({
  env: "你的云开发环境ID", // 例如：cloud1-xxx
  traceUser: true
});
```

**获取环境ID：**
1. 打开微信开发者工具
2. 点击工具栏的 **云开发** 按钮
3. 在云开发控制台查看环境ID

## 🔍 常见问题

### 1. request:fail url not in domain list

**原因：** 未配置request域名白名单

**解决：**
- 开发测试：开发者工具中勾选"不校验合法域名"
- 正式发布：在小程序后台配置 `https://api.siliconflow.cn`

### 2. API调用失败，返回401

**原因：** API密钥配置错误或无效

**解决：**
1. 检查 `miniprogram/utils/ai-helper.js` 中的API密钥
2. 确认密钥有效且有足够余额
3. 重新生成密钥并替换

### 3. 生成超时或失败

**原因：** 
- 网络问题
- API服务繁忙
- 模型响应慢

**解决：**
1. 检查网络连接
2. 尝试更换模型（修改 `ai-helper.js` 中的 `model` 参数）
3. 使用降级方案（代码已内置）

### 4. 生成的内容不理想

**原因：** prompt设计或模型参数需要调整

**解决：**
1. 修改 `miniprogram/utils/ai-helper.js` 中的prompt
2. 调整 `temperature` 参数：
   - 降低（0.5-0.7）：更保守、准确
   - 提高（0.8-1.0）：更创意、随机
3. 调整 `max_tokens` 参数控制输出长度

### 5. API费用问题

**说明：** 硅基流动按调用次数和token数量计费

**建议：**
1. 注册时通常有免费额度
2. 监控API使用量
3. 可以在 `ai-helper.js` 中实现缓存机制减少调用
4. 添加用户输入长度限制

## 🎨 自定义配置

### 更换LLM模型

编辑 `miniprogram/utils/ai-helper.js`：

```javascript
// 文本生成模型选择
model: "Qwen/Qwen2.5-7B-Instruct",  // 快速，中文好
// model: "Qwen/Qwen2.5-72B-Instruct", // 更强，但慢且贵

// 图片生成模型选择
model: "Qwen/Qwen-Image",            // 国产，速度快
// model: "stabilityai/stable-diffusion-3-5-large", // 效果更好，但慢且贵
```

### 调整超时时间

```javascript
timeout: 30000  // 30秒，可根据需要调整
```

### 修改图片尺寸

```javascript
image_size: "1024x1024"  // 可选：512x512, 768x768, 1024x1024
```

### 降级方案配置

当LLM调用失败时，会自动使用降级方案：
- **文本降级：** 使用原始内容作为标题和摘要
- **图片降级：** 使用Unsplash免费图片

可以在 `ai-helper.js` 中自定义降级图片列表。
