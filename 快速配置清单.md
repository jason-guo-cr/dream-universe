# 快速配置清单 ✅

完成以下步骤即可开始使用梦境宇宙小程序：

---

## 1️⃣ 获取硅基流动API密钥

- 访问：https://cloud.siliconflow.cn/account/ak
- 注册并登录
- 创建API密钥
- 复制密钥备用

---

## 2️⃣ 配置API密钥

**文件：** `miniprogram/utils/ai-helper.js`

**修改第8行：**
```javascript
const SILICONFLOW_API_KEY = "你的API密钥";
```

---

## 3️⃣ 配置云开发环境ID

**文件：** `miniprogram/app.js`

**修改第3-5行：**
```javascript
wx.cloud.init({
  env: "你的云开发环境ID",  // 例如：cloud1-xxx
  traceUser: true
});
```

**获取环境ID：**
1. 打开微信开发者工具
2. 点击"云开发"按钮
3. 在控制台查看环境ID

---

## 4️⃣ 配置request域名白名单

### 开发测试（临时方案）

1. 微信开发者工具 → 右上角"详情"
2. "本地设置"
3. 勾选 ☑️ **不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**

### 正式发布（必须）

1. 登录 https://mp.weixin.qq.com/
2. 开发 → 开发管理 → 开发设置 → 服务器域名
3. 在"request合法域名"中添加：
   ```
   https://api.siliconflow.cn
   ```
4. 保存配置

---

## 5️⃣ 上传云函数

1. 打开微信开发者工具
2. 右键点击 `cloudfunctions/dreamFunctions` 文件夹
3. 选择 **上传并部署：云端安装依赖**
4. 等待上传完成（约1-2分钟）

---

## 6️⃣ 运行测试

1. 点击"编译"按钮
2. 在模拟器中点击"记录梦境"
3. 输入梦境内容（至少10字）
4. 点击"生成"
5. 等待30-45秒
6. 查看生成结果

---

## ✅ 配置完成

如果以上步骤都顺利完成，恭喜你！可以开始使用了。

---

## 🐛 遇到问题？

### request:fail url not in domain list
→ 检查步骤4，配置域名白名单

### API调用失败，401错误
→ 检查步骤2，API密钥是否正确

### 云函数调用失败
→ 检查步骤3和步骤5，环境ID和云函数上传

### 生成很慢或超时
→ 正常现象，首次调用AI模型较慢，耐心等待
