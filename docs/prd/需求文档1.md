# 1. 项目说明

1. 开发一个和梦境相关的小程序，中文名为：梦宙编年史，英文名为：Dream Universe
2. docs目录为文档记录的目录，其中prd是我给你的需求文档目录，你的文档也统一输出到docs目录下

# 2. 功能描述

## 1.1 梦境时间线（首页）

1. 首页放一个《老友记》金句：No one care about you，but the universe does, and that says a lot. 
2. 首页以时间线的形式展示梦境

## 1.2 梦境记录

1. 默认时间是昨天，时间可以切换（同一天可以记录多条梦境）
2. 根据用户记录的内容调用LLM生成图片（可以先调用LLM根据用户的记录来优化生成图片的提示词）
3. LLM根据梦境内容自行决定生成多少张图片（最多三张）

## 1.3 A公解梦

1. A公（AI）解梦，类比周公解梦
2. 在梦境时间线页面可以选择一条或者多条梦境（最多三条）调用LLM结合梦境内容进行解梦，给用户展示LLM对梦境解析后的内容

# 3. 技术需求

1. 用户的数据需要隔离，每个用户只能看到自己的梦境数据
2. 需要实现小程序登录的能力（https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/login.html），系统中需要记录用户信息，用户表需要记录OpenID，如果是第一次使用则给一个随机昵称，用户可以自行修改昵称
3. 使用微信小程序云开发，使用云函数，云函数的超时时间配置为900s
4. 使用腾讯云clould base能力开发，包括数据存储、云存储（图片）、LLM能力
